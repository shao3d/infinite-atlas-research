# Infinite Atlas — Колода «КОСМОС» (дизайн-документ v1.0)

> Парная к `01-engagement-foundations.md` и `02-mechanic-library.md`. Здесь — конкретная колода из **16 карт** на тему «Космос», спроектированная как **эталон высокого reuse**: одно окружение (чернота + параллакс-звёзды + шарики-тела) и 2-3 library-движка несут массу карт почти даром, а 4-5 hero концентрируют дорогой «вау».
>
> Принцип сборки (из §4 библиотеки): `КОЛОДА = 4-5 HERO + ~3 LIBRARY-движка + МЕТА-СЛОЙ + REWARD + ГИД`. Главное оружие против «везде одно и то же» — **VARY THE VERB** (принцип 10): меняем ГЛАГОЛ и ТЕМП, а не окружение. Каждая карта проходит **ГЕЙТ A** (право на провал + ставка/таймер + личный рекорд) и **ГЕЙТ B** (authoring-cost честно).

---

## 1. Где Космос на спектре reuse: ЭТАЛОН ВЫСОКОГО REUSE

Космос — **правый край спектра «высокий reuse»** (противоположность динозаврам/Риму, где окружение надо варьировать). Причины, почему именно здесь окружение можно НЕ менять:

1. **Окружение объективно однородно.** Чернота + точечные звёзды + сферическое тело в центре — это и есть реальный космос. Ребёнок не ждёт смены декораций между планетами; смена декораций была бы даже неправдой. Сравните с динозаврами, где джунгли-болото-пустыня обязаны меняться, иначе фальшь и скука.
2. **Тела параметризуются дёшево.** 15+ небесных тел = один арт-движок «сфера + текстурная карта + кольца/атмосфера/спутники как слои». Это чистый gen-арт сферы (надёжная зона AI) + вектор-оверлей подписей. Per-card-стоимость арта = центы.
3. **Технически-точное (орбиты, гравитация, масштабы) рисует КОД детерминированно** (принцип 14 + §6 границы AI). Орбита, n-body, парабола гравманёвра — это математика, не хрупкий image-AI. Космос почти весь состоит из «технически-точного», поэтому он маршрутизируется в процедурку = $0/карта.

**НО** (честно по гейту B и принципу 10): однородность окружения — это и есть ловушка монотонности. Если все 16 карт — «покрути планету и почитай факт», колода = цифровая энциклопедия. Reuse мы берём **в ОКРУЖЕНИИ и ДВИЖКАХ**, а вариативность гоним **через ГЛАГОЛ и ТЕМП** (см. §10 boredom_solution). Это и есть точное применение «VARY THE VERB, NOT THE SETTING»: окружение одно, глаголы — спас/разрушь/подкрался/разгони/построил/запусти — все разные.

## 2. Shared core — ОДНО переиспользуемое окружение + связка движков

**Окружение (несёт визуал ВСЕХ 16 карт):**

- **Чёрный космос-фон + 2.5D-параллакс звёздного поля** (~$0, код; принцип 14, «2.5D-параллакс из 1 арта»). Глубина и «живость» без видео.
- **Сфера-тело в центре**: один gen-арт сферы на тело (надёжная зона AI), кольца/атмосфера/спутники — отдельными слоями. 15+ тел = 15+ дешёвых текстур на одном движке рендера.
- **Вектор-оверлей** для подписей, стрелок, орбитальных линий, шкал (принцип 15: текст НЕ в картинке).

**Связка движков, несущая массу (детали в §3):**

1. **Parametric Orbit Sandbox** — n-body/кеплеровы орбиты кодом. Один числовой движок закрывает «движение в космосе» для множества карт.
2. **Scale-as-Action (предскажи-и-проверь масштаб)** — Scale-Comparison поднят из «reward-штриха» до активного предсказания (по §2 библиотеки Scale нельзя как самостоятельную пассивную; даём predict-and-reveal + ставку).
3. **Slingshot / Aim-and-Launch** — параболический физ-движок гравманёвра, целься-запусти.

Эти три кода + окружение покрывают ~62% карт (10 из 16). Остальное — 5 hero + 1 мета-карта (альбом).

## 3. Library-механики (масса 60-70% карт)

### Движок A — Parametric Orbit Sandbox (#34 Parametric Sandbox, в library-режиме как Two-State/Trace-варианты + #3 Match+)

Один кеплеров/n-body движок: тела на орбитах, ползунки массы/скорости/расстояния, кнопка «пуск» и предсказание исхода. В library-режиме — упрощённые, параметризованные данными конфигурации (полный sandbox-hero — cap 1, см. hero «Чёрная дыра»).

- **covers_share:** ~31% (5 карт из 16).
- **example_cards:**
  - «Поймай Луну на орбиту» (Match+ #3: спутник↔планета, верно → встаёт на орбиту, мимо → улетает в черноту со смешным «вжух»).
  - «Собери Солнечную систему по порядку от Солнца» (Reorder→Launch #5: перепутал порядок → планеты сталкиваются, смешной хаос; верно → система раскручивается).
  - «Удержи кольца Сатурна» (Balance/Two-State: тонкая настройка — слишком быстро кольца разлетаются).
  - «Проведи комету через пояс астероидов» (Trace-Guided-Path #27: веди по орбите, сгорела → смешной пшик).
  - «Останови Землю — что будет?» (Two-State Diorama #13 + Quiz-in-Consequence встроенно: предскажи, выключи орбиту → последствие на экране).
- **why:** «движение в космосе» — это ровно то, что код делает детерминированно и бесплатно (принцип 14). Один движок орбит = много глаголов (поймай/упорядочь/удержи/проведи/предскажи) на одном окружении. Это сердце экономики космос-колоды.

### Движок B — Scale-as-Action: предскажи-и-проверь масштаб (#Scale поднятый через predict-and-reveal, обёрнут в #7 Catch / #4 Sort-Under-Pressure)

Scale-Comparison сам по себе — пассивная инфографика (§2 переклассификация). Поднимаем до игры: ребёнок **предсказывает** (перетаскивает ползунок «во сколько раз больше?» / ловит правильный размер), потом reveal с prediction error (принцип 5 curiosity gap).

- **covers_share:** ~19% (3 карты из 16).
- **example_cards:**
  - «Угадай, во сколько Юпитер больше Земли» (drag-ползунок предсказания → reveal, рекорд точности; ошибся сильно → Земля смешно теряется внутри Юпитера).
  - «Сортировка тел по размеру под метеоритным дождём» (Sort-Under-Pressure #4: тела летят потоком, успей разложить от карлика к гиганту до переполнения).
  - «Поймай только тела тяжелее Земли» (Catch/Dodge #7: лови верные по гравитации, поймал лёгкое → штраф-звук).
- **why:** масштаб космоса — главный интуитивный «вау» темы, но только если он **заслужен предсказанием** (hands-on + minds-on, §1.3). Дёшево (данные + ползунок/спрайты), даёт два возрастных слоя: 7-8 — сюрприз размера, 11-12 — оценка кратности (§1.6).

### Движок C — Slingshot / Aim-and-Launch гравманёвр (#22)

Параболический физ-движок: оттяни, целься, запусти зонд гравитационным манёвром мимо тел в цель.

- **covers_share:** ~12% (2 карты из 16).
- **example_cards:**
  - «Запусти зонд к Юпитеру гравиманёвром» (используй гравитацию планеты как рогатку; промах → зонд улетает в пустоту, рекорд по точности).
  - «Сбей курс — попади в кольцо Сатурна» (целься сквозь притяжение, провал виден).
- **why:** Angry-Birds-петля «прицел→мах→попал» (§3 #22) — чистый адреналиновый глагол «запусти», ноль gen-стоимости (физика + спрайты), встроенный провал и рекорд. Контраст-темп к тихим картам.

**Итого library: 5 + 3 + 2 = 10 карт (62%).** Доля «разбери/упорядочи» (Reorder, Sort, Match) = 3 карты ≈ 19% — **под лимитом ≤30%** (гейт A). Остальная масса — активные глаголы (поймай, проведи, удержи, запусти, предскажи).

## 4. Hero-карты (4-5 эмоциональных пиков + чейзы)

Жёсткие правила: видео hard cap 4-5/колода; Parametric Sandbox cap 1/колода; Build-and-Defend cap 1/колода. Берём **5 hero**, из них только **3 несут i2v-видео** (Солнце, Сверхновая, Чёрная дыра) — под капом и с запасом.

### HERO 1 — СОЛНЦЕ: «Останови вспышку» (Save-on-Time #28 + Destroy reveal)
- **mechanic:** Save-on-Time (#28, якорь эмоции колоды) — солнечная вспышка/корональный выброс летит к Земле, успей развернуть магнитный щит / увести спутники за таймер.
- **emotional_wow:** «Я спас Землю от солнечной бури» — эпизодическая память с максимальной ставкой (§1.2). За этой эмоцией ребёнок возвращается бить рекорд.
- **why_hero:** Save-on-Time несёт наибольшую эмоциональную ставку (интуиция Степана, §3 #28). Солнце — самый эмоционально заряженный объект колоды.
- **cost_note:** ядро процедурное ($0, симуляция НЕ видео). Дорогой gen оправдан в **одном hero i2v-клипе вспышки** (~$0.42-0.56) как cliffhanger-хук перед действием (#45, палец сразу в деле). Арт Солнца — 1 gen-сфера + шейдер пульсации.

### HERO 2 — САТУРН: «Построй кольца» (Drag-to-Assemble→Alive #15 + Balance #21)
- **mechanic:** Assemble→Alive (#15) — собери кольца из льда и камней по слоям-щелям (щель Кассини), финал ОЖИВАЕТ: кольца раскручиваются, играет свет. Старший слой — Balance: слишком много массы в щель → кольцо рушится.
- **emotional_wow:** «Я построил Сатурн своими руками» — авторство + эффект IKEA (§1.1 компетентность). Тихий, медитативный пик-контраст к шумному Солнцу.
- **why_hero:** Сатурн — визуальный «чейз» колоды (самый красивый объект), оправдывает богатую сборку. Эмоция авторства, а не адреналина — разнообразит палитру чувств.
- **cost_note:** БЕЗ видео. Дорогой gen — 1 премиум-арт Сатурна с кольцами (фольга-чейз на физ-карте). Слоты колец = human-разметка (authoring по гейту B), но движок Assemble переиспользуется.

### HERO 3 — ЧЁРНАЯ ДЫРА: «Поэкспериментируй с гравитацией» (Parametric Sandbox / Run-Sim #34, CAP 1)
- **mechanic:** Parametric Sandbox (#34, единственный на колоду) — двигай массу/расстояние, предскажи, что засосёт за горизонт событий, запусти симуляцию, смотри спагеттификацию и линзирование.
- **emotional_wow:** «Я скормил звезду чёрной дыре и увидел, как она растянулась» — запретный кайф + причинно-следственная магия (§1.5 prediction error). Возврат за экспериментом «а что если...».
- **why_hero:** Чёрная дыра — идеальный носитель guided-play sandbox: явление контринтуитивно, prediction error огромный. Это hero-слой для 11-12 (гипотезы, §1.6).
- **cost_note:** sim-код (готовая упрощённая модель линзирования, §3 #34 «брать готовые модельки»). 1 hero i2v-клип засасывания как награда-reveal (~$0.5). Reuse модели между темами = low (честно по гейту B) — поэтому cap 1.

### HERO 4 — СВЕРХНОВАЯ: «Взорви звезду» (Destroy / Demolish #32)
- **mechanic:** Destroy (#32) — доведи звезду до коллапса (убери опору ядра), запусти взрыв, смотри ударную волну и reveal: из чего звезда состояла внутри (слои), что разлетелось в новые элементы.
- **emotional_wow:** «Я взорвал целую звезду» — запретный кайф разрушения + «как устроено» через разбор (§3 #32). Огромный «ещё разок».
- **why_hero:** Destroy несёт самый громкий темп колоды и обучает структуре звезды через деструкцию (intrinsic integration). Идеальный чейз — редкое, мощное событие.
- **cost_note:** процедурка обломков/волны надёжна ($0). 1 hero i2v-клип вспышки сверхновой (~$0.5). Слои-внутренности = спрайты (дёшево).

### HERO 5 — МАРС: «Сядь на Марс» (Steer / Drive #36 — НОСИТЕЛЬ ГИДА, intro-карта)
- **mechanic:** Steer/Drive (#36) — рули спускаемым аппаратом сквозь атмосферу и пыльную бурю, погаси скорость, сядь в кратер. Гид-марсоход выкатывается из аппарата → знакомство с серией.
- **emotional_wow:** «Я посадил аппарат на другую планету и встретил своего ровера» — присутствие, контроль + эмоциональная привязка к гиду (§4.5).
- **why_hero:** Steer — КЛЮЧЕВАЯ для сшивания серии (§3 #36). Это intro-карта, где появляется гид Орбит/Ровер и тянет к следующей колоде. БЕЗ видео — экономим капы.
- **cost_note:** гид — ОДИН пайплайн-ассет (rig, реген запрещён). Трасса спуска/параллакс пыли из gen-арта ~$0. Без i2v.

**Видео-учёт: 3 hero-клипа (Солнце, Чёрная дыра, Сверхновая) < cap 4-5.** Сатурн и Марс — без видео.

## 5. Guide character — сквозной персонаж-гид

**Имя:** **ОРБИ** (зонд-исследователь с маленьким напарником-ровером «Пиксель»).

**Роль:** автономный космический зонд с любопытными «глазами»-объективами и солнечными парусами-крыльями. На Марсе из него выкатывается мини-ровер Пиксель. Орби — носитель механики **Steer/Drive** (#36): ребёнок рулит им в intro-карте и видит его в reward-слое КАЖДОЙ карты («Орби сфотографировал тебя у Сатурна — в альбом!»).

**Как сшивает серию и тянет к следующей колоде:**
- **Один ассет-rig** (character bible + 2.5D-rig), детерминированно во всех картах и всех будущих колодах (§3 #36, реген image-AI запрещён). Это центр retention-стратегии.
- **Эмоциональная привязка важнее уникальности карт** (§4.5 Dog Man): «хочу пройти весь космос с Орби» сильнее суммы интерактивов.
- **Daily-Surprise** от лица Орби (§4.1 #41): «Орби нашёл сегодня новый астероид — лови!».
- **Мост к следующей колоде:** финал альбома — Орби получает координаты следующего пункта путешествия (напр. «сигнал из глубин океана» → колода «Океан», или «странные кости на Марсе» → «Динозавры»). Гид физически переезжает в следующую тему — прямой драйвер «папа, купи следующую».

## 6. Collection-meta — коллекционный мета-слой

**Механика:** Collection-Album / Unlock-Secret (#40) + Daily-Surprise (#41).

- **Альбом «Карта Солнечной системы» N/N (16/16):** каждая пройденная карта зажигает своё тело на общей орбитальной карте. Видимые пустые слоты = прямой pester-power драйвер (§1.9, §4.1).
- **Чейзы (variable-ratio, без тёмных FOMO):** hero-карты — редкие «фольга-чейзы» в рознице (Солнце/Сатурн/Чёрная дыра/Сверхновая/Марс). Сатурн с фольгой колец — главный коллекционный магнит на полке (§4.6 hero фольгой).
- **Разблокировка секрета:** собрал все 16 → открывается **тайная 17-я «карта-секрет»**: пролёт зонда Орби за пределы Солнечной системы (Вояджер-момент, «золотая пластинка») + координаты следующей колоды. Секрет нельзя достать иначе как собрав весь сет → двигатель завершения.
- **Скрытый объект в каждой карте:** в каждой карте спрятан мини-астероид/спутник; найдёшь во всех 16 → бонус-чейз в альбом (микро-коллекция поверх макро).
- **Этика:** прогресс видимый и достижимый, hero как редкие, но честные чейзы (гейт A #40, без тёмных паттернов).

## 10. Boredom solution — как именно решена «везде одно и то же»

Окружение одно (чернота+звёзды+сфера) — значит, вся анти-монотонность держится на **смене ГЛАГОЛА и ТЕМПА** (принцип 10 VARY THE VERB + §4 правило выбора). План вариативности по 16 картам:

**Карта глаголов (16 разных эмоций-действий, ни один архетип не повторяется подряд):**
повёл → поймал → спас → предсказал → разрушил → упорядочил → подкрался → запустил → построил → разобрал → поэкспериментировал → разогнал → провёл → прицелился → удержал → собрал.

**Карта темпа (шумно↔тихо, намеренное чередование, §3 #31/#10):**
```
#1 средний  → #2 ТИХИЙ → #3 ГРОМКИЙ → #4 тихий → #5 ГРОМКИЙ → #6 средний →
#7 ТИХИЙ(stealth) → #8 громкий → #9 тихий → #10 ГРОМКИЙ → #11 средний →
#12 громкий → #13 тихий → #14 громкий → #15 тихий → #16 финал
```
Нигде нет двух громких или двух тихих подряд более чем на короткий стык; самые громкие пики (Солнце #3, Сверхновая #5, Сорт-под-дождём #10) разнесены тихими картами (Scale #4, подкрадись к комете #7, построй кольца #9).

**Три механизма против монотонности:**
1. **Палитра ЭМОЦИЙ, не только глаголов:** адреналин (Save, Destroy, Slingshot) ↔ медитация авторства (Assemble Сатурна, Trace кометы) ↔ любопытство-эксперимент (Sandbox ЧД, Scale-predict) ↔ тихое напряжение (подкрадись к комете #7). Ребёнок проживает 4 разных типа чувств на одном фоне.
2. **Два возрастных слоя на карту** (§1.6): 7-8 ловят сюрприз (размер Юпитера, взрыв), 11-12 — гипотезы (что засосёт ЧД, кратность масштаба) — одна карта не «выдыхается» по возрасту.
3. **Тихая карта-контраст #7 «подкрадись к комете»** специально вставлена как held-breath геймплей среди шумных — это vary-the-verb на уровне ТЕМПА, прямой приём из §3 #31.

**Итог:** монотонность космоса нейтрализована не сменой декораций (которой и не должно быть), а тем, что **каждая следующая карта просит другое действие, другой темп и другую эмоцию** — при этом движки и окружение переиспользуются, удерживая стоимость почти на нуле.
