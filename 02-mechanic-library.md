# Infinite Atlas — Библиотека интерактивных механик (ревизия v1.0)

> Финальная библиотека после двух адверсариальных критик (kid-engagement скептик + production-cost скептик). Парная к `01-engagement-foundations.md`. Здесь — **что строить**: какие механики дёшевы и переиспользуемы (library), какие оправданно дороги (hero), как параметризовать на много карт, и где AI хрупок.
>
> **Что изменилось против черновика:**
> 1. Введены два жёстких правила-гейта (см. ниже) — без них половина library остаётся «скучными кнопками».
> 2. Переклассифицированы cost/reuse у ~12 механик, где оценка была нереалистичной (процедурный ≠ бесплатный; very-high reuse → high из-за authoring-cost; very-low cost → low из-за мульти-состояниевой когерентности image-AI).
> 3. Добавлено 8 недостающих вовлекающих механик (разрушение, versus, escape, stealth, альбом-разблокировка, co-op с родителем, daily-surprise + дешёвые наполнители).
> 4. Скучные «открывалки» (X-Ray, Zoom, Scale, Spot-the-Difference, Coloring) переделаны или понижены до reward-слоя.

---

## 0. ДВА ПРАВИЛА-ГЕЙТА (применять к КАЖДОЙ карте перед публикацией)

### ГЕЙТ A — «Право на провал» (от kid-engagement критики)

> **Любая library-карта публикуется ТОЛЬКО если имеет: (а) таймер ИЛИ наступающую угрозу; (б) видимое ПЛОХОЕ последствие ошибки — не «мягкий отскок», а «убежало / сломалось / смешно лопнуло»; (в) личный рекорд.**
> Дополнительно: **доля «разбери/упорядочи»-механик (Sort/Match/Reorder/Quiz/Listen-Identify) в колоде ≤ 30%.** Иначе колода по ощущению = цифровая рабочая тетрадь, даже при разных глаголах.

### ГЕЙТ B — «Authoring-cost честно» (от production-cost критики)

> **Считать две отдельные метрики: engine-cost (пишется один раз) и authoring-cost per card (уровни, физика, граф, бит-карта, координаты, фактчек, локализация EU+US).** «very-high reuse» оставлять только там, где per-card = чистые данные/спрайты без дизайн-прохода и QA.
> Сквозные пайплайн-инварианты:
> - **Гид — ОДИН ассет** (character bible + sprite-sheet / 2.5D-rig), НЕ реген на карту. Любой реген гида image-AI запрещён.
> - **Видео — hard cap 4–5 hero/колода.** Engine-reuse плеера = very-high, ASSET-reuse видео = none.
> - **Мульти-состояниевые наборы** (слои X-Ray, стадии Grow, до/после) — через img2img-цепочку от одного базового арта + human-выравнивание, НЕ независимые генерации.
> - **Текст/подписи/схемы** — векторным/HTML-оверлеем поверх gen-фона, не в картинке.

## 3. КАРТОЧКИ МЕХАНИК

Формат: **Глаголы взаимодействия · Чем запоминается · Где AI хрупок · Как параметризуется · Гейты**.

---

### ГРУППА I. LIBRARY — дешёвая масса (чистые данные/спрайты)

#### 1. Tap-to-Sequence / Progressive Hotspots (Хотспоты-тур)
Сверхдешёвый интерактивный «тур по картинке»: на одном gen-арте 3–5 невидимых хотспотов, тап раскрывает мини-анимацию/факт/звук.
- **Глаголы:** тап по скрытым точкам сцены → reveal.
- **Запоминается:** curiosity gap «что под этим?» на каждой точке; агентность исследования.
- **AI хрупок:** почти нет — 1 атмосферный арт (надёжно) + координаты + TTS.
- **Параметризация:** 1 картинка + список хотспотов (координата → анимация/звук/факт).
- **Гейт A:** дать лёгкую ставку — «найди все 5 секретов до того, как гид уплывёт» + рекорд по времени. Иначе пассивно.

#### 2. Memory-Flip / Concentration (Парный переворот)
Классический memory-match на сетке: переворачивай карточки, ищи пары существ/планет/символов.
- **Глаголы:** тап-переворот, поиск пары.
- **Запоминается:** узнавание + повтор (сильный мнемонический эффект); встроенный рекорд.
- **AI хрупок:** ноль — иконки генерятся поштучно надёжно.
- **Параметризация:** набор иконок + размер сетки.
- **Гейт A:** таймер/ограничение ходов + рекорд. Менее «школьно», чем Match-the-Pairs со связями.

#### 3. Match-the-Pairs+ (Оживающие пары)
Соедини A↔B, но **каждое верное соединение ЧТО-ТО ОЖИВЛЯЕТ**: планета↔спутник — спутник встаёт на орбиту; нерв↔мышца — мышца дёргается; ошибся — «короткое замыкание», искра, смешной звук.
- **Глаголы:** тап по двум элементам / drag-линия.
- **Запоминается:** связь закрепляется повтором + оживлением (не галочкой).
- **AI хрупок:** почти нет — список пар + иконки.
- **Параметризация:** список пар + анимация оживления на верное.
- **Гейт A:** **таймер ОБЯЗАТЕЛЕН** (Match Madness) + комбо за серию без ошибок. Без оживления и таймера → выкинуть (это Anki-карточки).

#### 4. Sort-Under-Pressure (Сортировка-конвейер)
Не «разложи по корзинам спокойно», а **конвейер-под-давлением**: объекты едут/падают/прибывают сами, успей рассортировать поток до переполнения (стиль сортировки багажа / Papa's).
- **Глаголы:** drag в корзину-категорию на скорость.
- **Запоминается:** класс объекта через активное решение под цейтнотом; адреналин.
- **AI хрупок:** минимум — N корзин, M объектов, карта правильности + иконки.
- **Параметризация:** категории + объекты + скорость разгона.
- **Гейт A:** разгон скорости; **ошибка = объект портится/убегает/звучит фейл (видимо)**; рекорд «18 за 30 сек». Для старших — «спорный объект» (всеядный — куда?).

#### 5. Reorder→Launch (Порядок + пуск)
Не «расставь шаги по порядку» с галочкой, а **собери последовательность и НАЖМИ СТАРТ**: неправильный порядок = ракета взрывается на старте (смешно), метаморфоз — смешной мутант-франкенштейн; правильный = взлёт/превращение.
- **Глаголы:** drag элементов в ряд → кнопка «пуск».
- **Запоминается:** порядок проверяется ИСХОДОМ (сработало/рвануло), не оценкой; curiosity gap «а что если перепутать?».
- **AI хрупок:** почти нет — упорядоченный список + иконки.
- **Параметризация:** последовательность + анимация успеха/провала.
- **Гейт A:** провал = эффектный смешной взрыв/мутант (не страшно). Слит с Run-the-Simulation.

#### 6. Hidden-Object-Live (Найди убегающее)
Объекты не статичны: рыбки заметили тебя и удирают — успей тапнуть пока видно; таймер дыхания дайвера-гида («воздух кончается»).
- **Глаголы:** скан сцены + тап по убегающей цели; pinch-зум.
- **Запоминается:** активный поиск + живая ставка (скрылось — счёт ниже); связь с гидом-каталогом.
- **AI хрупок:** требует **слоистого рендера** — фон и объекты-цели РАЗДЕЛЬНО (иначе рандомизация позиций и реиграбельность фейковые). Ручная разметка координат + плейтест видимости.
- **Параметризация:** фон + набор спрайтов-целей + позиции.
- **Гейт A:** таймер дыхания + промах = существо скрылось; рекорд по находкам/времени. Факт — через характерное действие найденного при тапе, не текстом.

#### 7. Catch / Dodge (Лови/уворачивайся)
Двигай корзину/существо, лови нужное и избегай опасного — лови планктон для кита, уворачивайся от астероидов.
- **Глаголы:** drag по горизонтали; тайминг ловли; уклонение.
- **Запоминается:** аркадный азарт + рекорд; «правильная добыча» = факт темы (intrinsic).
- **AI хрупок:** ноль — классический catch-движок + спрайты «хорошее/плохое».
- **Параметризация:** спрайты + скорость + правила «хорошее/плохое».
- **Гейт A:** уже встроены провал (поймал плохое) и рекорд.

#### 8. Whack-on-Time (Реактивный тап)
Цели выскакивают, тапай правильные (накорми хищника верной добычей / собери нужные звёзды), избегай неверных, на таймер с рекордом. Дешевле Catch/Dodge (нет физики падения).
- **Глаголы:** реактивный тап по появляющимся целям.
- **Запоминается:** адреналин + рекорд; «что правильно тапать» = факт.
- **AI хрупок:** ноль — спрайты + спавн-логика.
- **Параметризация:** набор «правильных/неправильных» спрайтов + темп.
- **Гейт A:** тап неверного = штраф/фейл-звук; рекорд. Хороший контраст-глагол к drag-механикам.

#### 9. Quiz-in-Consequence (Развилка-в-механике) — СЛУЖЕБНОЕ
> **Запрещено как самостоятельная карта.** Разрешено ТОЛЬКО как развилка ВНУТРИ другой механики, где неправильный выбор имеет **немедленное плохое последствие на экране**: выбрал не ту еду удильщику — он уплыл голодный, сцена темнеет; повёл зонд не туда — сгорел.
- **Глаголы:** тап по 1 из 3 (иконки для младших).
- **Гейт A:** **обратный таймер (3-2-1, существо вот-вот сделает что-то само)** + болезненный (нестрашный) провал. Без таймера и провала — выкинуть.
- **AI хрупок:** почти нет (данные), НО скрыт authoring+фактчек+локализация налог. Использовать **библиотеку из 5–6 готовых параметризуемых анимаций-последствий** (дверь открылась / существо спасено / мост достроился), не рисовать на карту.

#### 10. Listen-as-Tool (Звук-инструмент, не квиз)
Не «послушай и ткни источник» (это аудио-квиз), а **звук как инструмент выживания/поиска**: «в темноте кто-то рычит — определи по звуку, безопасно ли подходить» (ошибся — существо прыгает, скример-лайт, смех-испуг).
- **Глаголы:** play → решение-действие под таймером.
- **Запоминается:** звук явления = сильный якорь, отдельный от визуала; ставка «выживу/нет».
- **AI хрупок:** ноль — TTS/SFX из Kling дёшевы.
- **Параметризация:** аудиоклип + безопасный/опасный исход.
- **Гейт A:** таймер «пока существо не уплыло» + провал с juice.

#### 11. Soundboard / Make-the-Sound (Извлеки звук)
Чисто экспрессивно-исследовательский звуковой движок: тапай части существа/прибора/инструмента — каждая издаёт свой звук, собери «мелодию».
- **Глаголы:** тап по зонам → звук; повтори паттерн.
- **Запоминается:** мультисенсорный якорь; «я сыграл».
- **AI хрупок:** ноль — фон + зоны + sfx/TTS.
- **Параметризация:** карта зон → звуков.
- **Гейт A:** мягкий — добавить мини-цель «повтори мелодию гида» с рекордом, иначе reward-уровень.

#### 12. Slider-Reveal / Drag-to-Compare (Шторка до/после)
UI-ползунок открывает второй слой поверх первого: день/ночь, здоровый/больной орган, руины/реконструкция. Дешевле и надёжнее Spot-the-Difference.
- **Глаголы:** drag ползунка-шторки.
- **Запоминается:** «найди что поменялось сам» + причина изменения при тапе различия.
- **AI хрупок:** 2 ВЫРОВНЕННЫХ кадра — **одним инпейнтом из базового**, не две генерации.
- **Параметризация:** 2 слоя + хотспоты-причины.
- **Гейт A:** добавить «верни как было» (восстанови руину тапами) для агентности + рекорд.

#### 13. Two-State Toggle Diorama (Переключи режим)
Один параллакс-арт с 2–3 переключаемыми состояниями через шейдер/слой-свитч (прилив/отлив, вулкан спит/извергается, день/ночь). Имитирует песочницу за ~$0.
- **Глаголы:** тап-переключатель режима; наблюдение перехода.
- **Запоминается:** ощущение причинности и агентности без дорогой симуляции.
- **AI хрупок:** надёжно — заранее отрендеренные состояния + переход.
- **Параметризация:** 2–3 состояния одной сцены.
- **Гейт A:** дать предсказание «что будет если включить вулкан?» + последствие. Дешёвая замена Parametric Sandbox для массы.

#### 14. Stamp / Sticker-Decorate Diorama (Наклей и оживи)
Drag готовых спрайтов-стикеров на фон-сцену, каждый при установке проигрывает микро-анимацию/звук. Дешевле Coloring (нет fill-зон).
- **Глаголы:** drag стикеров на сцену.
- **Запоминается:** эффект IKEA + шеринг с родителем.
- **AI хрупок:** надёжно — фон + набор спрайтов.
- **Параметризация:** фон + набор стикеров.
- **Гейт A:** мягкий (экспрессия) — связать с Photo-reward для шеринга. НЕ считать игровым драйвером массы.

### ГРУППА II. LIBRARY — движок + контент на тему (high reuse, есть authoring)

#### 15. Drag-to-Assemble→Alive (Собери и оживи)
Тащи детали и складывай целое — **но финал ОЖИВАЕТ**: собранный скелет встаёт и рычит, робот танцует.
- **Глаголы:** drag деталей в слот, snap, оживление на завершении.
- **Запоминается:** авторство сборки (из чего состоит) + награда движением.
- **AI хрупок:** генерить детали **не по отдельности, а нарезать из ОДНОГО консистентного арта** собранного объекта (1 проход → код режет по маске) — единый стиль/свет. Слоты-разметка = human-часы.
- **Параметризация:** набор деталей + слоты + анимация оживления.
- **Гейт A:** убрать подсветку слотов для старшего слоя; таймер («собери, пока музей не открылся»); кривая сборка = смешной франкенштейн ковыляет.

#### 16. X-Ray-Seek (Вскрой и найди)
Не «три и смотри», а **найди под слоями спрятанное** (срастили с Hidden-Object): где сломанная кость? где паразит? где трещина в коре?
- **Глаголы:** свайп-стирание маски + поиск цели; pinch между слоями.
- **Запоминается:** «я докопался до кости» — открытие как личное действие с целью.
- **AI хрупок:** **cost=low (не very-low):** послойные изображения одного объекта = пиксельная регистрация + анатомическая когерентность → img2img от базового + human-выравнивание. Точная анатомия (скелет/органы) для hero — **вектором кодом поверх gen-фона**, не image-AI.
- **Параметризация:** 3–5 слоёв + спрятанная цель (рандомизируемая).
- **Гейт A:** ограниченный «заряд рентгена» (выбирай ГДЕ светить) + таймер + рандом находки для реиграбельности.

#### 17. Zoom-Quest (Зум с челленджем)
На КАЖДОМ уровне зума — микро-задача-действие: на уровне клетки поймай вирус, на ДНК почини разрыв, на галактике найди свою планету пока не уехала. Зум = способ перемещения между мини-челленджами.
- **Глаголы:** pinch-зум + действие на уровне.
- **Запоминается:** иерархия масштаба через действие, а не слайд-шоу.
- **AI хрупок:** уровни-картинки надёжны атмосферно; точные микро-структуры стилизовать. Когерентность уровней = img2img-цепочка.
- **Параметризация:** 3–5 уровней + челлендж на каждом.
- **Гейт A:** «что прячется глубже?» надо ЗАСЛУЖИТЬ действием. Без действия — понизить до reward.

#### 18. Feed-and-React (Покорми, смотри хаос)
**Неправильная еда — самая интересная:** скорми хищнику салат — выплюнет с отвращением; перекорми — отрыжка/раздуется/лопнет (запретный кайф «а если ещё?»). Реакция на ВСЁ, не только на правильное.
- **Глаголы:** drag объекта в зону-рот; наблюдение цепочки; повтор.
- **Запоминается:** причина-следствие + телесный юмор + эксперимент.
- **AI хрупок:** **видео убрать из дефолта** — реакция спрайтовой/процедурной анимацией (частицы/морф/шейдер), не i2v на каждую карту.
- **Параметризация:** набор еды + реакции; голодное существо слабеет на таймере.
- **Гейт A:** мини-ставка (голод на таймере) + «дневник реакций» (коллекционный микро-драйв).

#### 19. Rhythm / Timing-Tap (Ритм)
Тапай в ритм — в удары сердца, пульс пульсара, гребки триремы, шаги динозавра.
- **Глаголы:** ритмичный тап в окна-биты; удержание длинных нот; скоринг.
- **Запоминается:** ритм явления кинестетически; success-juice на каждом попадании; рекорд.
- **AI хрупок:** ноль — бит-карта + tap-окна + скоринг код; аудио из TTS/Kling/SFX.
- **Параметризация:** трек + бит-карта + спрайт.
- **Гейт A:** скоринг точности + рекорд встроены.

#### 20. Connect-the-Circuit→Alive (Замкни и оживи)
Тяни связи между узлами под ставкой: «робот теряет питание / сердце остановилось — замкни цепь за 10 сек, оживи». Перегрузка при лишнем соединении (искры, дым).
- **Глаголы:** drag-связь от узла к узлу; проверка; эффектное оживление.
- **Запоминается:** логика связей через решение под угрозой + оживление системы.
- **AI хрупок:** **reuse=medium:** валидный граф + проверка разрешимости + защита от тупика + **вектор-узлы (не image-AI на схемы)**. Узлы — из переиспользуемой вектор-библиотеки.
- **Параметризация:** граф узлов/связей + несколько решений.
- **Гейт A:** таймер-угроза + перегрузка (смешной фейл) + эффектное оживление.

#### 21. Balance / Stack (Баланс и стопка)
Ставь объекты, удерживая равновесие/строя стопку — нагрузи рычаг, сложи камни акведука. Дженга-драма резкого конца.
- **Глаголы:** drag объектов на платформу/рычаг; точная установка.
- **Запоминается:** физика равновесия через риск обрушения (эмоциональный пик).
- **AI хрупок:** **reuse=medium-high:** тюнинг масс/центров/порога обрушения + плейтест «на грани» + **фикс физ-шага (fixed timestep)** для воспроизводимости рекордов.
- **Параметризация:** объекты/массы/сцена (генератор раскладок по уровню риска).
- **Гейт A:** обрушение = резкий конец игры (драма) + рекорд.

#### 22. Slingshot / Aim-and-Launch (Прицелься)
Оттягивай и отпускай, целясь — запусти зонд гравитационным манёвром, камень из баллисты.
- **Глаголы:** drag-оттягивание (сила/угол), отпускание; коррекция.
- **Запоминается:** Angry-Birds-петля «прицел→мах→попал»; мастерство траектории; рекорд.
- **AI хрупок:** ноль — параболический физ-движок + цели.
- **Параметризация:** снаряд/мишени/гравитация.
- **Гейт A:** провал (промах) + рекорд встроены.

#### 23. Excavate→Reconstruct (Раскопай, собери)
Смахивай песок кистью-пальцем, находи фрагменты, собирай находку — роль палеонтолога/археолога. Объединяет X-Ray (маска) + Assemble (слоты).
- **Глаголы:** свайп-смахивание → drag-сборка фрагментов.
- **Запоминается:** «я откопал и собрал это сам»; сильная связка с гидом-археологом.
- **AI хрупок:** арт находки надёжен; оба под-паттерна надёжны.
- **Параметризация:** находка + фрагменты + порядок.
- **Гейт A:** таймер раскопок + рекорд; неполная сборка = смешной результат.

#### 24. Grow / Lifecycle (Вырасти)
Ухаживай и продвигай существо по стадиям — яйцо→динозавр, головастик→лягушка, звезда→сверхновая.
- **Глаголы:** тап/drag ухода (полить/покормить/согреть) → триггер стадии.
- **Запоминается:** последовательность стадий через заботу + предвкушение (Tamagotchi-привязанность).
- **AI хрупок:** **reuse=medium-high:** консистентность ОДНОГО существа между стадиями = img2img-цепочка от базового дизайна (фикс палитры/стиля) + human-отбор, не независимые генерации.
- **Параметризация:** арт стадий + условия перехода.
- **Гейт A:** существо грустит/слабеет если не ухаживать (ставка) + возврат к «питомцу».

#### 25. Mix / Combine (Смешай)
Тащи и комбинируй элементы — вода+солнце→пар, газ+пыль→планета. Little Alchemy-эффект.
- **Глаголы:** drag элемента на элемент → reveal результата.
- **Запоминается:** реакции через эксперимент + сюрприз; коллекционирование рецептов.
- **AI хрупок:** ноль — таблица A+B=C + иконки поштучно.
- **Параметризация:** таблица рецептов.
- **Гейт A:** реиграбельность через поиск всех комбинаций (коллекционный микро-драйв) + «дневник открытий».

#### 26. Operate-the-Machine (Поуправляй прибором)
Жми кнопки/тяни рычаги/крути ручки — пульт ракеты, навести телескоп, перископ.
- **Глаголы:** тап кнопок, drag рычагов, крутящие жесты → эффект на состоянии.
- **Запоминается:** «я оператор/пилот»; назначение контролов через управление.
- **AI хрупок:** **cost=low ТОЛЬКО с UI-библиотекой:** панель контролов = вектор-вёрстка + хитбоксы + состояния (image-AI не тянет приборную панель с подписями). Собрать переиспользуемую библиотеку (тумблер/слайдер/крутилка/лампа/стрелка); фон — gen-арт.
- **Параметризация:** компоновка контролов + маппинг на эффекты.
- **Гейт A:** дать цель/последствие управления + мини-провал (перегрев/авария).

#### 27. Trace-Guided-Path (Веди по заданному пути)
Дешёвый вариант: веди объект по заранее заданному инвариантному пути (валидация = расстояние до полилинии). Контент = одна кривая на карту.
- **Глаголы:** свайп вдоль пути; запуск объекта.
- **Запоминается:** кинестетическое авторство «довёл».
- **AI хрупок:** ноль — путь предзадан, валидация по расстоянию; фон gen-арт.
- **Параметризация:** одна кривая-путь + цель.
- **Гейт A:** ставка (комета сгорит / кровь не дойдёт — пациент бледнеет на таймере) + звёзды за скорость. (Свободная физика-версия — см. #37, hero.)

### ГРУППА III. HERO / BOTH — эмоциональные пики (cap-лимиты)

#### 28. Save-on-Time (Спаси стаю) — ЯКОРЬ ЭМОЦИИ КОЛОДЫ
Оттаскивай существ от опасности на таймер — рыбок от фонарика удильщика, где факт-угроза = механика. **Интуиция Степана.**
- **Глаголы:** drag-оттаскивание из опасной зоны; преграда между угрозой и жертвой; гонка с таймером.
- **Запоминается:** ЭТАЛОН эпизодической памяти «я спас стаю» — эмоция+авторство+ставка.
- **AI хрупок:** ядро процедурное ($0); хрупкость только в спрайтах угрозы/жертвы. **Реализовать симуляцией, НЕ видео.**
- **Параметризация:** угроза + жертвы + таймер (упрощённый MID-вариант для library, богатый — hero).
- **Гейт A:** провал (не успел — съели) + рекорд времени. Концентрировать дорогой «вау» здесь.

#### 29. Escape / Chase (Спасайся сам) ★НОВАЯ
«T-Rex заметил тебя — беги, сворачивай, прячься за камень за 5 сек». Спаси СЕБЯ — сильнейший адреналин для 10-летки (наследник удильщика, но от первого лица).
- **Глаголы:** swipe-уклонение/повороты/прятки на время.
- **Запоминается:** «меня чуть не сожрал тираннозавр» — железная эпизодическая память; палец сразу в деле (rule of first seconds).
- **AI хрупок:** ноль — чистый код + спрайты + параллакс.
- **Параметризация:** преследователь + препятствия + дистанция.
- **Гейт A:** провал (поймали) + рекорд дистанции/времени, реиграбельно.

#### 30. Versus / Beat-the-Creature (Сразись) ★НОВАЯ
Соревнование «ты против существа» — двигатель №1 для 8–12. «Перетяни канат с T-Rex» (тапай быстрее укуса), «обгони гепарда» (ритм-бег), «задержи дыхание дольше кита» (held-tap).
- **Глаголы:** быстрый тап / held-tap / ритм-бег против существа-противника.
- **Запоминается:** существо = противник с характером; факт зашит в правила (кит ныряет 90 мин — вот почему проиграешь); рематч «в этот раз сделаю».
- **AI хрупок:** ноль — код + спрайты.
- **Параметризация:** существо + параметр соревнования + порог.
- **Гейт A:** проигрыш + рекорд + рематч встроены.

#### 31. Stealth / Don't-Wake (Подкрадись) ★НОВАЯ
«Сфотографируй спящего T-Rex, не разбудив» / «проплыви мимо удильщика в темноте». Напряжение тишины, held-breath геймплей — контраст к шумным механикам (vary-the-verb на уровне ТЕМПА).
- **Глаголы:** медленный точный drag, «замри» (отпусти при опасности).
- **Запоминается:** тихо-страшное напряжение; факт зашит (ночные/хищные повадки).
- **AI хрупок:** ноль — код + спрайты + параллакс.
- **Параметризация:** существо + зоны опасности + порог пробуждения.
- **Гейт A:** проснулся → рык → мгновенный провал с juice; рекорд.

#### 32. Destroy / Demolish (Разрушь) ★НОВАЯ
Запретный кайф разрушения + обучение через деструкцию. Метеор в планету (кратер и волна), снеси римскую стену катапультой и увидь кладку внутри, взорви звезду в сверхновую.
- **Глаголы:** прицел/запуск/удар; наблюдение разрушения и того, что ВНУТРИ.
- **Запоминается:** «как устроено» через разбор на части; огромный «ещё разок».
- **AI хрупок:** процедурка надёжна (физика обломков); спрайты слоёв.
- **Параметризация:** объект + слои-внутренности + сила.
- **Гейт A:** скоринг разрушения + reveal внутреннего устройства (intrinsic).

#### 33. Build-and-Defend (Tower-Defense lite) — HERO, cap 1/колода
Расставляй защитников/преграды против волн — защити гнездо динозавра, риф от хищников.
- **Глаголы:** тап-размещение защитников; drag преград; наблюдение волн.
- **Запоминается:** «я отбил волну, спас гнездо» — авторство+ставка+исход; стратегический слой для старших.
- **AI хрупок:** TD-движок код; **баланс волн = высокая скрытая стоимость геймдизайна** → вынести в датадривен-конфиг с авто-симуляцией прохождения (бот прогоняет волны).
- **Параметризация:** юниты + волны + сцена.
- **Гейт A:** проигрыш (гнездо разорено) встроен. **hero-only, не размазывать.**

#### 34. Parametric Sandbox / Run-Simulation (объединено) — HERO, cap 1/колода
Двигай солнце/температуру/гравитацию → наблюдай реакцию; опц. «пуск» с предсказанием исхода. **Run-Simulation слит сюда** (флаг наличия явного триггера).
- **Глаголы:** drag ползунков/объектов → (опц. пуск) → наблюдение.
- **Запоминается:** причинно-следственные связи через собственный эксперимент (ядро guided play); гипотезы для старших.
- **AI хрупок:** **reuse=low между темами:** каждая симуляция — отдельная численная модель + педагогическая валидация поведения. Брать ГОТОВЫЕ упрощённые модельки (n-body, клеточный климат), не писать с нуля.
- **Параметризация:** ВНУТРИ темы (параметры), между темами — новая модель.
- **Гейт A:** предсказание перед пуском (prediction error) + сюрприз результата. **Cap 1/колода.**

#### 35. Reaction-Chain (Цепная реакция / Руб Голдберг) — HERO
Расставляй элементы и запускай каскад — солнце→тает лёд→поднимается вода.
- **Глаголы:** drag-расстановка триггеров → пуск → наблюдение каскада.
- **Запоминается:** системное мышление через построение и запуск собственной цепи.
- **AI хрупок:** **reuse=low по контенту:** каждая цепь = уникальный левел-дизайн + QA разрешимости. Дать набор переиспользуемых типовых триггеров (нагрев/толчок/рычаг) с гарантированной физикой; **авто-тест «цепь доходит до конца»** перед публикацией.
- **Параметризация:** набор блоков-триггеров + целевой каскад.
- **Гейт A:** «сработало по цепочке!» + провал (цепь оборвалась). Ограничить числом/колода.

#### 36. Steer / Drive — НОСИТЕЛЬ ГИДА — HERO (intro-карта)
Рули пальцем/наклоном, ведя гида по маршруту — дайвер сквозь риф, марсоход. **КЛЮЧЕВОЙ для сшивания серии персонажем.**
- **Глаголы:** drag-руление / наклон; уклонение/сбор по пути.
- **Запоминается:** присутствие и контроль; гид-персонаж = эмоциональная привязка к серии.
- **AI хрупок:** **ГИД — ОДИН пайплайновый ассет** (character bible + sprite-sheet / 2.5D-rig), используется детерминированно во всех картах и колодах. **Реген гида image-AI запрещён** (центр retention-стратегии «купи следующую»). Трассы/параллакс из gen-арта почти бесплатны.
- **Параметризация:** трасса + препятствия (гид неизменен).
- **Гейт A:** провал (врезался/потерялся) + рекорд; гид появляется в reward-слое каждой карты.

#### 37. Trace-Free-Physics (Свободная физика) — HERO
Нарисуй ЛЮБУЮ рампу, которая доведёт комету мимо планет в цель — много решений, агентность «я придумал».
- **Глаголы:** свайп-рисование рампы/линии; запуск объекта по физике.
- **Запоминается:** кинестетическое авторство + «получилось по-своему».
- **AI хрупок:** физ-солвер = код (hero-нагрузка авторинга/QA).
- **Параметризация:** стартовые условия + цель + препятствия.
- **Гейт A:** провал видимый (комета сгорела) + звёзды за изящество/скорость («а можно короче?»).

#### 38. Tilt-to-Control + touch-fallback (Наклон)
Наклоняй телефон, чтобы катить объект — лаву в вулкане, шарик-планету по лабиринту.
- **Глаголы:** физический наклон (акселерометр); **fallback на drag**, если сенсор недоступен.
- **Запоминается:** физическое свойство (гравитация) проживается телом.
- **AI хрупок:** **reuse=low-medium:** per-card level design не амортизируется + хрупкость гироскопа в PWA (iOS требует жеста-разрешения, ломается в ландшафте). Уровни — **процедурная сборка лабиринта из тайлов по сид-данным**, не рисовать руками.
- **Параметризация:** раскладка тайлов-препятствий + цель.
- **Гейт A:** обязательный touch-fallback (иначе карта мертва на части устройств) + ставка/таймер (лава догоняет).

#### 39. Pour / Pseudo-Flow (Налей поток)
Наклоняй/тащи, чтобы лить и заполнять — наполни акведук, заполни лёгкие. Сильно для Рима (акведуки) и тела (кровь/воздух).
- **Глаголы:** наклон/drag для направления потока; открытие каналов; заполнение до цели.
- **Запоминается:** принцип потока через тактильное наполнение + головоломка русла.
- **AI хрупок:** **НЕ настоящая CFD** (роняет FPS) — сеточная/клеточная псевдо-жидкость или поток-по-каналам. **reuse=low** + perf-флаг (профилировать на нижнем устройстве как gate).
- **Параметризация:** каналы/русла + цель заполнения.
- **Гейт A:** дефицит + таймер (вода высыхает / пациент бледнеет); перелив = потоп (смешной фейл).

### ГРУППА IV. МЕТА-СЛОЙ И REWARD (двигатели «купи следующую»)

#### 40. Collection-Album / Unlock-Secret ★НОВАЯ — КРИТИЧЕСКАЯ ДЛЯ БИЗНЕСА
Явная механика-носитель коллекционного мета-слоя (раньше был только принципом): альбом N/N с пустыми слотами, прогресс к 100%, и **СЕКРЕТ, разблокируемый только когда собрал всю колоду** (или нашёл скрытый объект в каждой карте). «Соберёшь всех 5 hero — откроется тайный 6-й».
- **Глаголы:** просмотр альбома; тап по слотам; разблокировка по сбору.
- **Запоминается:** видимый незакрытый сет = прямая причина выпрашивать недостающие карты.
- **AI хрупок:** ноль — данные + иконки.
- **Параметризация:** структура альбома + условие секрета.
- **Гейт A:** variable-ratio (hero как редкие чейзы), без тёмных FOMO; прогресс видимый и достижимый.

#### 41. Daily-Surprise / Variable-Reward ★НОВАЯ
Причина ВЕРНУТЬСЯ к той же карте завтра: «сегодня твой динозавр в новом настроении / нашёл новый предмет / гид прислал задание дня». Дофаминовый крючок возврата без тёмных паттернов.
- **Глаголы:** заход → сюрприз дня → микро-действие.
- **Запоминается:** ощущение «живого» компаньона; поднимает retention/LTV.
- **AI хрупок:** почти ноль — рандом из заранее заготовленных данных/спрайтов.
- **Параметризация:** пул сюрпризов на карту/гида.
- **Гейт A:** variable-ratio этично; даёт возврат для 7–8, которым «побей рекорд» слабовато.

#### 42. Co-op-with-Parent (Игра вдвоём за одним экраном) ★НОВАЯ
«Держи телефон вдвоём»: один наклоняет — другой тапает; родитель читает факт-подсказку — ребёнок действует. Превращает родителя из плательщика в со-игрока → охотнее купит следующую.
- **Глаголы:** разделённые роли на одном экране.
- **Запоминается:** ценно для retail-репокупки и сарафана/шеринга.
- **AI хрупок:** ноль — компоновка существующих ассетов + данные.
- **Параметризация:** роли + подсказки.
- **Гейт A:** опциональный режим поверх существующих механик.

#### 43. Coloring→Alive + AR (Раскрась, оживи) — REWARD-СПУТНИК
Два режима чётко: (1) свободный «мой динозавр» БЕЗ правильности → сразу в AR/Photo для шеринга; (2) «оживи раскрашенное»: закрасил → динозавр ВСТАЁТ и ходит в твоей раскраске.
- **Глаголы:** тап-заливка зон + палитра; drag декора.
- **Запоминается:** «мой собственный» + эффект IKEA + сюрприз-оживление за труд.
- **AI хрупок:** контур + fill-зоны надёжны; движок заливки тривиален.
- **Параметризация:** контур + зоны.
- **Гейт A:** научный факт — как РАЗБЛОКИРОВКА («реальный был таким — хочешь увидеть?»), не как оценка. **НЕ считать одним из ~3 library-движков массы** — это спутник к Photo/AR.

#### 44. Photo-Frame → AR (Сними реальность) — REWARD, PoC-ГЕЙТ
Общий вирусный финал-награда. **По умолчанию — деградированная фото-рамка** (спрайт-PNG-оверлей поверх статичного кадра камеры, без plane-tracking) — даёт 80% эффекта «в моей комнате» за 10% сложности, работает офлайн везде.
- **Глаголы:** наведение камеры, тап-снимок, drag размещения, сохранение/шеринг.
- **Запоминается:** «динозавр был у меня в комнате» + шеринг → прямой триггер «купи следующую».
- **AI хрупок:** **технический риск HIGH:** настоящий AR (WebXR/plane-detection) в офлайн-PWA на iOS фактически недоступен → **PoC-гейт ДО планирования**; альфа-край спрайта как у видео.
- **Параметризация:** спрайт-объект на все колоды (один движок).
- **Гейт A:** связать с Coloring (свой раскрашенный объект в кадре). Настоящий AR — только если PoC подтвердит на целевых устройствах.

### ГРУППА V. СЛУЖЕБНОЕ (никогда не главный интерактив)

#### 45. Tap-to-Reveal→Cliffhanger (Хук-затравка)
Тап оживляет карту (рёв, взрыв) как 2–3-сек крючок. **ЭТО «скучные кнопки» POC, если оставить голым.**
- **Глаголы:** один тап → оживление, обрывающееся на cliffhanger.
- **Запоминается:** само по себе слабо (пассивно); ценность — открывает curiosity gap.
- **AI хрупок:** хромакей-край на меху/перьях/пыли/огне/полупрозрачности = halo (human-чек кадров); i2v не амортизируется (~$0.42–0.56/клип + ре-роллы).
- **Параметризация:** **engine-reuse=very-high, ASSET-reuse=none.** Видео — **hard cap 4–5 hero/колода**; для library «оживление» — 2.5D-параллакс + процедурные шейдеры (дыхание/моргание/блик/пар), почти $0.
- **Гейт A:** **запретить тап-ради-тапа.** Первое касание = первое действие механики, ИЛИ хук обрывается на cliffhanger, передающем управление («T-Rex замечает тебя — БЕГИ», палец уже действует). Параллакс-колыхание само по себе слишком слабо как отдельный хук.

## 4. АРХИТЕКТУРА КОЛОДЫ (15–30 карт) — как собирать

```
КОЛОДА = 4–5 HERO + ~3 LIBRARY-движка (масса) + МЕТА-СЛОЙ + REWARD + ГИД
```

**1. HERO (4–5 карт) — эмоциональные пики и редкие чейзы.** Брать из групп СПАСИ/ЗАЩИТИ/VERSUS/РАЗРУШЬ/ЭКСПЕРИМЕНТ: Save-on-Time, Escape/Chase, Versus, Destroy, Build-and-Defend (cap 1), Parametric Sandbox (cap 1), Reaction-Chain, Steer (intro-гид). Несут «вау» и оправдывают цену колоды.

**2. ~3 LIBRARY-движка — масса 60–70% карт**, параметризуемых данными. Брать из дешёвой массы (группа I) + движков с high reuse (группа II). **НО:** доля «разбери/упорядочи» (Sort/Match/Reorder/Quiz/Listen) ≤ 30% — остальное закрывать активными глаголами (Catch, Whack, Hidden-Live, Feed, Rhythm, Assemble→Alive).

**3. МЕТА-СЛОЙ — Collection-Album + Daily-Surprise** сшивают серию и тянут «купи следующую».

**4. REWARD — Photo-Frame (+Coloring→Alive)** общий на все колоды, вирусный шеринг.

**5. ГИД (Steer-rig)** — один ассет, появляется в reward каждой карты.

### Правило выбора механики под карту (против «везде одно и то же»)

1. Сначала задать **ЭМОЦИЮ-ГЛАГОЛ** карты (спас/построил/открыл/вырастил/разрушил/разогнал/победил/подкрался) → выбрать группу.
2. **Не ставить две карты подряд на одном архетипе** — чередовать глаголы даже на одном фоне (решает и космос-однообразие, и динозавров/Рим).
3. Чередовать и **ТЕМП**: шумные (Escape, Versus, Destroy) ↔ тихие (Stealth, Hidden-Live, Soundboard).
4. **Маршрутизация по AI-надёжности:** технически-точное (анатомия, механизмы, орбиты, схемы) → процедурные группы/код + вектор-оверлей; существа/атмосфера/«красиво издалека» → gen-арт+параллакс+видео; подписи/стрелки → вектор-оверлей.

## 5. ГРУППИРОВКА ПО ГЛАГОЛУ-ДРАЙВЕРУ (выбирать механику под эмоцию карты)

| Глагол-драйвер | Эмоция | Механики | Роль |
|---|---|---|---|
| **СПАСИ/ЗАЩИТИ** | забота + ставка (сильнейшая память) | Save-on-Time, Build-and-Defend | hero-пики |
| **СПАСАЙСЯ/СРАЗИСЬ/ПОДКРАДИСЬ** ★ | адреналин, versus, тишина | Escape/Chase, Versus, Stealth | hero/both, возврат 10-летки |
| **РАЗРУШЬ** ★ | запретный кайф + «как устроено» | Destroy/Demolish | hero, огромный «ещё разок» |
| **ПОСТРОЙ/СОБЕРИ** | агентность, эффект IKEA | Assemble→Alive, Connect→Alive, Balance, Excavate, Reaction-Chain | «я сделал это» |
| **ОТКРОЙ/ВСКРОЙ** | curiosity gap + магия скрытого | X-Ray-Seek, Zoom-Quest, Hidden-Live, Slider-Reveal, Tap-Hotspots | любопытство (с целью-действием!) |
| **УПРАВЛЯЙ/ВЕДИ** | присутствие, контроль, гид | Steer (гид), Operate-Machine, Tilt, Slingshot, Trace, Pour | Steer сшивает серию |
| **ЭКСПЕРИМЕНТИРУЙ** | guided play, гипотезы | Parametric Sandbox/Run-Sim, Mix, Two-State Diorama | hero-кандидаты |
| **ВЫРАСТИ/ИЗМЕНИ** | забота + прогресс | Grow/Lifecycle, Feed-and-React | Tamagotchi-привязанность |
| **РАЗБЕРИ/УПОРЯДОЧИ** | классификация (≤30%!) | Sort-Pressure, Match+, Reorder→Launch, Memory-Flip, Quiz (служебн.) | дешёвая масса, **под гейтом A** |
| **РИТМ/СЕНСОРИКА** | кинестетика, слух | Rhythm, Listen-as-Tool, Soundboard, Catch, Whack | тело и слух как каналы памяти |
| **ЭКСПРЕССИЯ/ФИНАЛ** | самовыражение + вирус | Coloring→Alive, Stamp-Diorama, Photo-Frame | шеринг → «купи следующую» |
| **МЕТА** | коллекция + возврат | Collection-Album, Daily-Surprise, Co-op-Parent | двигатели бизнеса |
| **ХУК** | «вау» 2–3 сек | Tap-to-Reveal→Cliffhanger | **никогда голым** |

## 6. ОСЬ СТОИМОСТИ (для маршрутизации бюджета колоды)

- **VERY-LOW cost / VERY-HIGH reuse (масса, ~$0/карта, чистый код/данные):** Tap-Hotspots, Memory-Flip, Match+, Sort-Pressure, Reorder→Launch, Catch, Whack, Quiz (служебн.), Stamp-Diorama, Coloring, Collection-Album, Daily-Surprise, Listen-as-Tool, Soundboard.
- **LOW cost / HIGH reuse (движок + контент на тему):** Hidden-Live, Slider-Reveal, Two-State Diorama, Assemble→Alive, X-Ray-Seek, Feed-React, Rhythm, Slingshot, Excavate, Mix, Trace-Guided, Escape, Versus, Stealth, Co-op-Parent.
- **LOW–MEDIUM cost / MEDIUM reuse (есть authoring/QA на карту):** Zoom-Quest, Connect→Alive, Balance, Grow, Operate-Machine.
- **MEDIUM cost / LOW–MEDIUM reuse (hero-кандидаты, арт/уровни/симуляция уникальны):** Save-on-Time, Destroy, Build-and-Defend (cap 1), Parametric Sandbox/Run-Sim (cap 1, reuse low между темами), Reaction-Chain, Steer, Tilt, Pour, Trace-Free-Physics.
- **ОТДЕЛЬНО (технический риск / не амортизируется):** Photo/AR (PoC-гейт, conditional), Tap-to-Reveal видео (asset-reuse none, hard cap 4–5/колода).
